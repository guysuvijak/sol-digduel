## ğŸ“‚ Installation & Setup

To run this project locally, follow these steps:

### **1. Clone the repository**

```bash
git clone https://github.com/guysuvijak/digduel.git
cd digduel
```

### **2. Install dependencies**

```bash
npm install --force
or
bun install
or
pnpm install
```

### **3. Start the development server**

```bash
npm run dev
or
bun run dev
or
pnpm run dev
```

### **4. The app will be available**

[http://localhost:3000](http://localhost:3000)

### **5. Export to cross-platform**

```bash
npm run tauri
or
bun run tauri
or
pnpm run tauri
```

note: If you want to Export Cross-platform, please follow requires setup before [Tauri Prerequisites](https://v2.tauri.app/start/prerequisites/)
